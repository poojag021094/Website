How to create a background image with transparency?

In order to create background image with transparency we need to:

intially set body and html height as 100% so that our image would cover the entire screen.

1) create a div whose height would be 100% i.e. height : 100%;
2) Set the background image of that div with format like background-image :url('your image url');
3) set background-size: cover so that our image would cover the entire div.
4) set backround-repeat: no-repeat in order to have only one uniform view of our image.
5) set background-position: center in order to center position our image on the screen.
6) In order to create transparency effect we make use of opacity attribute. i.e. opacity : .2 ans so on as per the requirement.
